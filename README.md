\# SteeleLab - Active Directory Home Lab



\## Overview

SteeleLab is a self-built virtual enterprise lab environment created using VMware Workstation and Windows Server 2022. The purpose of this project was to strengthen hands-on infrastructure, networking, Active Directory, file sharing, and application server support skills.



\## Demo Video



\[!\[Watch the demo](https://img.youtube.com/vi/BmoV4YQGpfo/0.jpg)](https://youtu.be/BmoV4YQGpfo)



\## Lab Environment

\- \*\*Domain Controller:\*\* SteeleLab

\- \*\*Domain:\*\* steelelab.local

\- \*\*File Server:\*\* FS1

\- \*\*Application Server:\*\* APP1

\- \*\*Platform:\*\* VMware Workstation

\- \*\*Operating System:\*\* Windows Server 2022



\## Network Configuration

| Server | Role | IP Address |

|---|---|---|

| SteeleLab | Domain Controller / DNS | 192.168.190.10 |

| FS1 | File Server | 192.168.190.20 |

| APP1 | IIS Application Server | 192.168.190.30 |



\## What I Built

\- Deployed a Windows Server 2022 Domain Controller

\- Configured static IP addressing and DNS

\- Joined FS1 and APP1 to the steelelab.local domain

\- Created a file server with shared departmental folders

\- Installed IIS on APP1

\- Built a basic internal web application landing page

\- Validated communication between servers
